## 1. 🥇 The "Top 20" Seed Catalog (The Winners)
Please request the data for the absolute **10 to 20 best-selling products** that we want to feature for the online launch.

For each product, we need:
- [ ] **High-Quality Photos:** At least 2-3 photos per product. (Clean background + lifestyle/in-room shots if possible).
- [ ] **Exact Pricing:** The retail price, plus any special online launch discounts.
- [ ] **The Specs:** Exact dimensions (Length x Width x Height), weight, and materials.
- [ ] **Descriptions:** A short, engaging paragraph describing the product.

## 2. 🚚 Logistics & Shipping Check
We need to know exactly how the company will handle online orders to build the shipping logic. Ask the team:
- [ ] "If someone buys a heavy sofa online tomorrow, how exactly do we ship it?"
- [ ] "How much does shipping cost us for heavy items vs. light items (like pillows)?"
- [ ] **Decision Needed:** Will we offer a Flat Rate, Free Shipping over $X amount, or Dynamic rates based on city?

## 3. 💳 Checkout & Payments Strategy (V1)
Please confirm with the stakeholders which checkout path we are taking for the initial launch:
- [ ] **Option A (Recommended for V1):** WhatsApp Checkout. The user builds a cart online and clicks "Comprar", which sends a perfectly formatted WhatsApp message to a sales rep to close the deal and arrange payment manually. (Fastest to launch, high-touch, zero gateway fees).
- [ ] **Option B:** Fully Automated E-Commerce. We integrate a Colombian payment gateway (Wompi, ePayco, MercadoPago) so the user pays with a credit card directly on the site.

## 4. 📦 Inventory Management
- [ ] Ask the team: "How will we know if an item sold online is actually in stock at the Restrepo store?" 
- [ ] **Decision Needed:** Will we have dedicated stock for the online store, or will someone manually update the website inventory daily based on physical store sales?
